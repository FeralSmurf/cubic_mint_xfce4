# cubic_mint_xfce4

1. The script runs chroot commands to create a custom ISO using cubic.
2. The script will ask the user to copy the casa_buna.jpg and icon.png into the chroot env. Just use drag and drop into cubic, then press "Copy".
3. Make the script executable with chmod +x my_chroot_commands_xfce4.
4. Run the bloody script and hope for the best with ./my_chroot_commands_xfce4.
5. Hopefully, we will get an iso that will install mint with a tailor-made theme (casa_buna.jpg wallpaper, icon.png start menu icon) and will have anydesk and teams in the bar. Steam install is also included.
6. Feel free to modify as you see fit.
